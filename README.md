University Administration and Application Management System (UAAMS) Industrial Consulting Project – QHO635 Project Overview

The University Administration and Application Management System (UAAMS) is a web-based platform designed to streamline and centralise the university admissions process for both students and university administrators.

Students can register, submit applications, upload required documents, and track application progress through a secure online portal. University administrators can review applications, manage student records, and issue offer or rejection decisions through a dedicated dashboard.

The project aims to replace fragmented manual workflows with a scalable digital solution that improves efficiency, communication, and user experience.
